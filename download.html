Server Setup

The server is necessary for the Car Thing to communicate with the Pi (and to avoid the "192.168.1.1 can't be reached" error). It also allows the Pi to connect to speakers, the microphone, GPS, and accelerometer, and adjust volume dynamically based on the accelerometer readings.

Download Server Files

Important Notes

  • Volume Adjustment: The volume is dynamically boosted relative to the accelerometer to ensure you can hear music while in motion. Keep this in mind if you’re using the Car Thing while driving, as volume may increase with speed.
  • iOS Users: We currently do not have access to a Mac, so iOS support is not yet available. Once we implement it, please note that you will need to approve the certificate for "PepperMarioYT" in your device settings to open the app. Alternatively, you can use AltStore to sideload the app. iOS has strict certificate policies, and this step is required to run the app.
